Doxygen Commands

Ausführliche Beschreibung auf: https://www.doxygen.nl/manual/commands.html

Für die Beschreibung der Elemente

  • @brief { brief description } - mit autobrief nicht nötig
  • @details { detailed description } - automatisch nach erster Leerzeile nach @brief

  • @author { list of authors }
  • @version { version number }
  • @date { date description }
  • @copyright { copyright description }

  • @remark { remark text }
  • @attention { attention text }
  • @note { text }
  • @warning { warning message }
  • @error { error text }
  • @see { references }
  • @since { text }
  • @test { paragraph describing a test case }
  • @par [(paragraph title)] { paragraph } - user defined paragraph

Eintrag in Listen:

  • @bug { bug description }
  • @todo { paragraph describing what is to be done }
  • @deprecated { description }
  • @xrefitem <key> "(heading)" "(list title)" { text } - generalization of commands such as @todo and @bug

  • @pre { description of the pprecondition }
  • @post { description of the postcondition }
  • @invariant { description of invariant }

  • @headerfile <header-file> [<header-name>] - header file that should be included for use this element, overrides default from parser

Steuerung ob Doxygen Code beachtet/verarbeitet wird

  • @cond [(section-label)]
  • @endcond

  • @if (section-label)
  • @ifnot (section-label)
  • @else
  • @elseif (section-label)
  • @endif

  • @noop ( text to be ignored )

  • @internal - INTERNAL_DOCS is set to YES
  • @endinternal

Gruppierung von Elementen

  • @name [(header)]
  • @{
  • @}


  • @defgroup <name> (group title)
  • @ingroup (<groupname> [<groupname> <groupname>])
  • @addtogroup <name> [(title)]
  • @weakgroup <name> [(title)]

Textformatierung

  • @a @e @em <word> - next word italic
  • @b <word> - next word bold
  • @c @p <word> - next word typewriter font
  • @li @arg { item-description } - list
  • @n - new line

  • @parblock
  • @endparblock

  • @emoji "name" - https://gist.github.com/rxaviers/7360908
  • @n - new line

  • @~[LanguageId] - english, german - sprachenabhängige Texte
