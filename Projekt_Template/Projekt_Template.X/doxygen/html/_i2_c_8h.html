Übergibt angeforderte I2C-Anfrage an das FIFO und liefert den akutellen Status.

Parameter
address7 Bit Adresse des Slaves
num_writeAnzahl der zu sendenen Bytes, bei 0 keine Write Zugriff
writebufZeiger auf zu schreibende Daten
num_readAnzahl der zu lesenden Bytes, bei 0 keine Read Zugriff
readbufZeiger auf Bereich, in welchem Daten abgespeichert werden sollen
statusZeiger, um akuellen Status zurückzugeben
callbackZeiger auf eine Callback-Funktion, welche nach Abschluss der Anfrage aufgerufen werden soll
IDID zur Identifikation der Anfrage
Rückgabewerte
1,Anforderungwurde angenommen, die FSM wird getriggert
0,FSMist beschäftig, Anforderung kann nicht angeommen werden

void initI2C ( void  )

Initialisiert die I2C-Kommunikation.

Aktiviert das I2C Modul und konfiguriert die Pins SDAx und SCLx als serielle PORT-Pins.
