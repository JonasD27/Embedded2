Interrupt des Timer1, welcher jede Sekunde ausgeführt wird. 

Damit wird jede Sekunde eine I2\+C-\/\+Anfrage für den Licht-\/ und den Temperatursensor gestellt. Als Erweiterung wird nun ein Zeiger auf die auszuführende Callback-\/\+Funktion sowie eine ID bei den Anfragen übergeben. 

Definiert in Zeile \mbox{\hyperlink{interrupts_8c_source_l00028}{28}} der Datei \mbox{\hyperlink{interrupts_8c_source}{interrupts.\+c}}.
